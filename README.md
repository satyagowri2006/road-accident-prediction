# Road Accident Prediction using XGBoost

## Overview
This project predicts road accident severity using the US Accidents dataset.

## Models Compared
- Random Forest (89.56%)
- XGBoost (94.03%)
- LightGBM (93.41%)
- Hybrid Model (93.90%)
- Tuned XGBoost (94.51%)

## Best Result
🏆 Tuned XGBoost achieved **94.51% accuracy**
